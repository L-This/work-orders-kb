# Work Orders Knowledge Base - مرجع أوامر العمل

مشروع Next.js جاهز للنشر على Vercel ومربوط بـ Supabase.

## الصفحات

- الصفحة الرئيسية: Dashboard + بحث عام بالبند.
- صفحة المشروع: بحث داخل مشروع واحد.
- صفحة الموقع: قصة الموقع + Timeline + أوامر العمل.
- صفحة الاستيراد: رفع Excel + ربط الأعمدة + معاينة + استيراد.

## متغيرات Vercel المطلوبة لاحقًا

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## ملاحظة

يمكن نشر الواجهة بدون المتغيرات أولًا للتأكد من نجاح البناء، ثم إضافة مفاتيح Supabase من Vercel لاحقًا.
