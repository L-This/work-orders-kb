# مرجع أوامر العمل — الاستيراد الذكي V3

هذه النسخة تضيف صفحة استيراد فعلية لملف جدول كميات مشروع بريمان الحالي.

## ما الذي تستخرجه الصفحة؟

- بيانات المشروع ورقمه وتواريخ العقد والمقاول.
- بنود جدول الكميات والكميات الأصلية والأسعار.
- أوامر العمل التي تحتوي بيانات فعلية فقط.
- مواقع التنفيذ المرتبطة بكل أمر عمل.
- الكميات المنفذة والرصيد الظاهر بعد كل أمر.
- سجل دفعة الاستيراد والصفوف الخام للرجوع إليها.

## طريقة التركيب

ارفع محتويات هذا المجلد إلى مستودع GitHub الحالي `work-orders-kb` مع استبدال الملفات، ثم انتظر نشر Vercel التلقائي.

المتغيرات المطلوبة في Vercel كما هي:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

بعد النشر افتح `استيراد Excel`، ارفع ملف بريمان، راجع النتائج، ثم اضغط `اعتماد واستيراد المشروع`.

## منع التكرار

يمكن إعادة استيراد نفس الملف بعد التعديل. النظام يحدّث المشروع والعقد وجدول الكميات وأوامر العمل والسطور الموجودة اعتمادًا على رقم المشروع ورقم أمر العمل ورقم صف المصدر، بدل إنشاء نسخ مكررة.
